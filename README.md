# 🌍 An Ethical and Open-Source Project
MaturAI was created with a simple belief: every student should have free access to high-quality educational tools.

For this reason, the project is fully open source. Students, teachers, and developers are free to study, modify, and adapt it to their own educational needs. The current version is configured around the subjects of Italian, English, Computer Science, and Computer Networks, but these are only the starting point.

MaturAI should not be seen as a finished product. Instead, it is a foundational platform on which countless oral examination experiences can be built. Different schools, countries, curricula, and educational contexts can customize the agents, subjects, evaluation criteria, and conversation flows to create entirely new simulations.

Every oral exam conducted through MaturAI can be unique. Every configuration can tell a different story. The goal is not to standardize learning, but to empower creativity, accessibility, and personalization in education.

I hope that, over time, a community of contributors will expand the platform, creating new examination models and learning experiences that are as diverse as the students who use them.

Education should be open. Preparation should be accessible. Learning should be personal. 🎓🤖✨

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
